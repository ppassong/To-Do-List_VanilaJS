## < 개발 후기 >
JavaScript에 대해서 좀 더 공부하고 알아보기 위해서 순수 자바스크립트. 바닐라 자바스크립트에 대한 클론 코딩을 진행하였다.<br>
각 기능들 사이에 체크해주고 전환해주는 함수 선언에 대한 중요성과 역할을 느꼈으며, 사용된 함수나 문법에 대해서 각각 따로따로 보면 이해는 되지만<br>
이것들을 스스로 제로부터 다 엮어서 스크립트를 코딩하라고 하면 확실히 어려울 것 같다는 느낌을 받았다.<br>
기존의 프레임워크나 라이브러리가 얼마나 좋은 효율성과 사용성을 부여해주는지 체감이 되기도 하고<br>
바닐라 JavaScript 사용이 JavaScript 실력을 끌어올리는데 도움이 많이 되겠다는 생각이 새삼 들었다.
